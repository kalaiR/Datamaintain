Overview API v2.0

Fixido provides API to import Leads, Order and to export leads as well. Like every other APIs, its a REST based API. Before you begin using the API, you must register an application to get the access key and secret key. To access REST API, you must have one access key and secret key. You can create an application here.

Once you created the application, system automatically generate access key and secret key for you, which you can see in the same page. System also automatically grant rights to few essential method such as create/edit/list Lead. You can review your rights in API Playground. You can create more applications with different roles

Technical Information

With every request to the API, you must attach the access key and secret key in POST data. And based on the method, you will also add some more data to POST. Since its REST based API, you can access it using any programming language which perform HTTP Request.

Now, Fixido API provides the data in JSON format (also supports JSONP). Response for all the request always follow the same structure. Response object has following properties,

  • status - boolean, true or false - Returns false whenever your request is failed to complete".
  • error - list of errors - List of errors which occurred while performing your request.
  • data - result object - Actual data for your request.

Sample Request and Response

Request:
http://fixido.com/api/leads/list/?access_key=[ACCESS_KEY]&access_secret=[SECRET_KEY]
Response:
        {
          'status':true,
          'error':[],
          'data':[
            {
              'title':'Lead Title',
              'description':'Lead Description'
            },
            {
              'title':'Another Lead Title',
              'description':'Another Lead Description'
            }
          ]
        }

Leads Create - creates a new lead

Use Leads Create to create a new lead in the Fixido marketplace system.

Make sure to provide as much information as possible to give the buyers a great lead.

You need to be a seller in Fixido in order to create a lead. If not, fill up the form in Fixido dashboard provided leads page.

Guideline for creating Leads

Leads Get - get your lead details

Use Leads Get to return details for a specified lead. With Leads Get you can only return your provided and bought leads

Input: Lead Id

Leads List - list your leads

Use Leads List to return a list of your provided leads. With Leads List you can only return your provided leads.

Leads Provided List - list your provided leads

Use Leads provided List to return a list of your provided leads. With Leads provided List you can only return your provided leads.

Leads Bought List - list your bought leads

Use Leads bought List to return a list of your bought leads. With Leads bought List you can only return your bought leads.

Leads Update - update your leads

Use Leads Update to update your leads. You can only update data in your provided leads.

Input: Lead Id

Leads Delete - delete your lead

Use Leads Delete to delete your leads. You can only delete any of your provided leads.

Input: Lead Id

Lead Buy - Buy a lead

Use Lead Buy to purchase a lead from Fixido. You can buy any leads that are available in Fixido.

Input:

Lead Id: Indicates the lead id that has to be purchased.

Quantity: by default it will be 1. Ignore this field.

Payment: by default it will be Fixido account.

Filters List - lists your filter

Use filters list to check the filters that are applicable/created by you.

Input:

Page: pagination. Specify the page number you need. Start with 1. Check the output to see how many pages exists.

Result_per_page: Indicates the number of results to be displayed in each page.

Categories List - list all lead categories

Use Categories List to get a list of all lead categories in the Fixido marketplace system.

Use the categories with the leads

Input:

language (ISO-639-1, two characters language codes; For ref: http://en.wikipedia.org/wiki/List_of_ISO_639-1_codes)

Page: pagination. Specify the page number you need. Start with 1. Check the output to see how many pages exists.

Result_per_page: Indicates the number of results to be displayed in each page.

Search Detail - Get details of the searched lead

Use Search Detail to return details for a specified lead. With Lead Get you can return data from any leads in the system. If you bought the lead, you will also see all locked contact details (like phone number and e-mail)

Input: Lead Id

Search List - Search for leads

Use Search List to get a list of leads. You specify search criteria and use pagination.

Inputs:

q = criteria (specify any search words you need. Leave blank to get all leads)

page = pagination. Specify the page number you need. Start with 1. Check the output to see how many pages exists in your result.

budget_start = What the consumer wants to spend on the product/service. Minimum.

budget_end = What the consumer wants to spend on the product/service. Maximum.

deal_start = When the consumer interest starts.

deal_end = When the consumer interest ends.

price_start = Lead price. Minimum.

price_end = Lead price. Maximum.

locations = Specify the location of the consumer.

Recently Search List - get your recent searches

Use recently search list to view the latest search done by you in Fixido. It lists all the search text by user by recent search text at top.

Recently Viewed Leads List - get recently viewed leads list

Use recently viewed leads to view the latest leads by you in Fixido. It lists all the leads viewed by user by recent leads at top.

Trade List - get trade lists

Use trade list to return details of trade for your lead. As a lead seller, it will be helpful for you to view the list of your leads sold in Fixido.
